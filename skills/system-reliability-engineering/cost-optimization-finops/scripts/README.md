### Budget Alerts

```bash
# Crear budget
python budget_alert.py create \
  --name monthly-budget \
  --amount 10000 \
  --period monthly \
  --thresholds 50,80,100 \
  --email ops@example.com

# Listar budgets
python budget_alert.py list

# Ver estado
python budget_alert.py status --name monthly-budget
```
